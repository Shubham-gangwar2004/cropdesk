# 🌾 CropDesk – Agricultural Marketplace Platform

## 🚀 Overview
**CropDesk** is a full-stack agricultural marketplace platform that connects farmers directly with dealers, eliminating middlemen and enabling fair and transparent trade. It allows farmers to list crops and dealers to browse, filter, and communicate in real-time.

---

## ✨ Features
- 🌱 Farmers can list crops with details (title, category, price, quality, location)
- 🔍 Dealers can search and filter crops easily
- 💬 Real-time chat system for direct communication
- 🔐 Secure authentication (login/signup for farmers & dealers)
- 📍 Location-based crop discovery
- 📱 Responsive and user-friendly UI

---

## 🛠️ Tech Stack
- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Real-time:** Socket.IO
